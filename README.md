## 关于华南农业博物馆筹建办公室资产管理系统

华南农业博物馆筹建办公室资产管理系统是用于华南农业博物馆筹建办公室进行固定资产管理和藏品管理的系统


## License

华南农业博物馆筹建办公室资产管理系统基于Laravel，同样使用[MIT license](http://opensource.org/licenses/MIT).
